A second demo is shown in the demo folder. This links to the send-mail.php file using a relative reference for the form action. Click on this link to go to the page.

Each of the form field types are shown in a box to make it easier to see them visually. These boxes are not required for display purposes.

The fields are organised in a side-by-side label:field arrangement.

Other layouts are possible by altering the sample css.

The hidden field will not have a label as it is not shown to the end user.

Two checkboxes that are grouped together

This checkbox is on its own, and not related to the other two.

Radio buttons are best served in at least pairs. The name should be the same for each with a different value that is sent back when the form is submitted.
